For Admob Integration:
Change YOUR_APPLICATION_ID in AndroidManifest.xml with your application id in admob:
<meta-data
            android:name="com.google.android.gms.ads.APPLICATION_ID"
            android:value="YOUR_APPLICATION_ID"/>
Also change the value of my_banner and my_ins in strings.xml:

<string name="my_banner">ca-app-pub-3940256099942544/6300978111</string>
<string name="my_ins">ca-app-pub-3940256099942544/1033173712</string>

By default it is a test AD for admob. Change this when ready for publishing.

For FirebaseAnalytics:
Upload your google-services.json file in app folder
